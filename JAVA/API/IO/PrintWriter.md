## java.io.PrintWriter

* `PrintWriter(String fileName)`

  构造一个将数据写入文件的 `PrintWriter` 。文件名由参数指定
  
* `PrintWriter(File file)`

    为指定的文件对象创建一个 `printwriter` 对象
    
* `void print(mixd mix)`

    将 `string`,`char`,`char[]`,`int`,`long`,`float`,`double`,`boolean` 写入文件中
    
* `void println()`

    打印一个换行
    
* `void printf()`

    格式化
    
* `void close()`

    关闭文件
