## java.lang.CharSequence

* `IntStream codePoints()`

  产生由当前字符串的所有 `Unicode` 码点构成的流

* `char charAt(int index)`

  返回给定索引处的码元

* `int length()`

  返回在这个序列中的码元的数量

* `CharSequence subSequence(int startIndex, int endIndex)`

  返回由存储在 `startIndex` 到 `endIndex -1` 处的所有码元构成的 `CharSequence`

* `String toString()`

  返回这个序列中所有码元构成的字符串
