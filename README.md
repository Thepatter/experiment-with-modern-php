#### 笔记

##### 规范

* 使用 markdown 进行写作
* 标题采用 h3，正文使用 h4 ~ h6
* 采用多行代码代替单行代码，尽量保留代码上下文
* 字母及数字两边空格
* 代码使用多行代码块标注语言
* 属性和结构使用斜体小写蛇形书写 *innodb_bool_pool_size*
* 类和接口采用首字母大写斜体 *Integer*
* 示例代码尽量采用 Java/C 书写
* 正文使用中文标点符号
